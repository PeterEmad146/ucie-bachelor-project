# UCIe Protocol Implementation in gem5

## Project Overview
Implementation of Universal Chiplet Interconnect Express (UCIe) protocol in gem5 simulator.

## Directory Structure 
- 'gem5/' - gem5 simulator source code
- 'src/' - UCIe implementation source files
- 'docs/' - Documentation and specifications
- 'tests/' - Test cases and validation
- 'benchmarks/' - Performance benchmarks
- 'results/' - Simulation results and analysis
